## Available Scripts

### Running the development server.

```bash
    npm run dev
```

### Building for production.

```bash
    npm run build:analyze
```

### Running the production server.

```bash
    npm run start
```

### Linting the code

```bash
    npm run lint
    npm run lint:style
```
